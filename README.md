## Environment setup
    - Node version 16 or later
    - laravel 11
    - Mysql
    
## Project setup
    - clone the project: git remote add origin https://github.com/Shahjalal7311/laravel_11_inretia_react.git
    - go to the dirctory 
    - composer install: composer install
    - npm install: install npm

## For deployement
  - php artisan serve
  - npm run dev
